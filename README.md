# 🚀 What I Can Do With Python

> Turning ideas into working tools...

---

## 📖 About This Project

This repository is a collection of practical Python projects built from curiosity, real-life needs, and continuous learning.

Instead of just learning concepts, I challenged myself to apply Python to solve problems, from simple everyday tasks to data analysis workflows and automation.

Each project in this series answers one question:

> *“What can I actually build with Python?”*

---

## 💡 The Idea Behind This Series

Learning Python is one thing. Using it to create useful tools is another.

_This series is my way of:_
- Practising consistently  
- Thinking in problems and solutions  
- Building real, usable tools  
- Documenting my growth publicly  

It’s not about perfection; it’s about **progress and practical application**.

---

## 🛠️ What You’ll Find Here

Across different episodes, I have built:

- 📊 Data visualisation tools (bar chart generators)  
- 🧮 Mathematical utilities (fraction calculator, number-to-words converter)  
- 🔤 Text-processing tools (word builder from letter combinations)  
- 📁 Data handling workflows (raw file processing)  
- 📈 Python-powered Excel User-Defined Functions (UDFs)  

Some are simple.  
Some are more involved.  
All are built with intention.
